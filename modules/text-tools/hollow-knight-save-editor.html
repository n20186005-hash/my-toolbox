Using the Editor

Welcome to the Hollow Knight Save Editor. This tool allows you to modify key values in your save file to recover lost items, fix bugs, or experiment with different builds.

Safety & Privacy

This tool runs entirely in your browser. Your save files are NOT uploaded to any server. All processing happens locally on your device, ensuring your data remains private.

  • Backup First: Always create a copy of your user1.dat file before editing. If something breaks, you can restore the backup.
  • JSON Support: This editor runs purely in your browser. It parses JSON structures commonly found in decrypted save exports or modded saves.

FAQ & Troubleshooting

Important Warning

Do not set values unreasonably high (e.g. 99999 Health) as this may crash the game UI. Stick to game-legal limits when possible.

Why can't I load my .dat file?

Original game files are encrypted (AES). You may need to use a save decryptor first, or use this tool with modded/JSON saves.

What is Geo?

Geo is the currency of Hallownest. You can edit this value to buy items from Sly or other vendors.
